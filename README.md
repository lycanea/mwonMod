> mwonmod is a lightweight Fabric mod that enhances your Melon King experience by providing configurable overlays, in-game shortcuts, and informational HUD elements :3

## Features:
- Ingame auction timer and flawless NPC spawn timer
- A simple toggleable inventory rundown overlay
- Keybinds to use pretty much any command (`@bank`, `@trinket [0-6]` etc)
- Sign upgrade preview tooltips
- Improved number formatting (commas)
- Desktop notifications for in-game events such as monarch change
- Performance improvements
- Many more QOL features

_This mod is made specifically for Melon King on Diamondfire, you likely won't find utility with it otherwise_

_If you have issues or feature requests, please make a github issue or contact me via discord!_
